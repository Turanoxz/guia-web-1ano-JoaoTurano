Os Três Pilares do Desenvolvimento Web

HTML: Estrutura e Semântica

HTML (HyperText Markup Language) é a linguagem que define a estrutura e o significado do conteúdo web:

<main class="conteudo-principal">
  <article class="post">
    <h1>Título do Post</h1>
    <p>Primeiro parágrafo...</p>
    <img src="foto.jpg" 
         alt="Descrição da imagem"
         loading="lazy">
  </article>
</main>
  • Define a estrutura do documento
  • Fornece significado ao conteúdo
  • Base para acessibilidade

CSS: Estilo e Layout

CSS (Cascading Style Sheets) controla a apresentação visual e o layout dos elementos:

/* Variáveis e layout moderno */
:root {
  --primary: #2563eb;
  --spacing: clamp(1rem, 2vw, 2rem);
}

.container {
  display: grid;
  grid-template-columns: 
    repeat(auto-fit, minmax(300px, 1fr));
  gap: var(--spacing);
  padding: var(--spacing);
}
  • Controla estilos e layout
  • Responsividade com Grid/Flex
  • Animações e transições

JavaScript: Interatividade

JavaScript torna as páginas interativas e dinâmicas:

// Módulo de tema escuro
const themeToggle = document.querySelector(
  '#theme-toggle'
);

function toggleTheme() {
  document.body.classList.toggle('dark');
  const isDark = document.body
    .classList.contains('dark');
  localStorage.setItem('theme', isDark);
}

themeToggle.addEventListener(
  'click', 
  toggleTheme
);
  • Adiciona interatividade
  • Manipula dados e DOM
  • Gerencia estado da aplicação
